# LumenUI - AI/LLM Integration Guide

> LumenUI (`@ryanhe919/lumen-ui`) is a modern React UI component library with Apple-like glassmorphism design, built with TypeScript and TailwindCSS v4.

## Quick Start

```bash
npm install @ryanhe919/lumen-ui
```

```tsx
// Import components and styles
import '@ryanhe919/lumen-ui/dist/style.css'
import { LMButton, LMInput, LMModal } from '@ryanhe919/lumen-ui'
```

### Peer Dependencies
- `react` >= 18
- `react-dom` >= 18

### Theme Setup
Wrap your app with a `data-theme` attribute. Available themes: `light` (default), `dark`, `blue`, `green`, `redWhite`.

```html
<div data-theme="dark">
  <App />
</div>
```

## Design System

### Size System
All components support 6 sizes via the `size` prop:
`xs | sm | md | lg | xl | 2xl` (default: `md`)

### Color Tokens (CSS Variables)
- Primary: `--lm-primary-50` to `--lm-primary-950` (Apple System Blue #007AFF)
- Semantic: `--lm-success-*`, `--lm-warning-*`, `--lm-error-*`, `--lm-info-*`
- Gray: `--lm-gray-50` to `--lm-gray-950`
- Background: `--lm-bg-default`, `--lm-bg-paper`, `--lm-bg-elevated`, `--lm-bg-hover`
- Text: `--lm-text-primary`, `--lm-text-secondary`, `--lm-text-tertiary`
- Border: `--lm-border-default`, `--lm-border-light`, `--lm-border-strong`
- Shadow: `--lm-shadow-sm`, `--lm-shadow-md`, `--lm-shadow-lg`, `--lm-shadow-xl`
- Radius: `--lm-radius-sm` (6px), `--lm-radius-md` (10px), `--lm-radius-lg` (12px), `--lm-radius-xl` (16px)
- Transitions: `--lm-transition-fast` (120ms), `--lm-transition-normal` (180ms), `--lm-transition-slow` (250ms)
- Easing: `--lm-ease-default`, `--lm-ease-spring`, `--lm-ease-out`, `--lm-ease-in-out`

### Feedback

#### LMMessage + LMMessageContainer
Toast notification system.

```tsx
const [messages, setMessages] = useState<LMMessageItem[]>([])

const showMessage = (type: LMMessageType, content: string) => {
  setMessages(prev => [...prev, { id: Date.now().toString(), type, content, duration: 3000 }])
}

<LMMessageContainer
  messages={messages}
  onClose={(id) => setMessages(prev => prev.filter(m => m.id !== id))}
  position="top-right"
/>
```

`LMMessageItem`: `{ id, type: 'success' | 'error' | 'warning' | 'info', title?, content, duration? }`
Position options: `'top-right' | 'top-left' | 'top-center' | 'bottom-right' | 'bottom-left' | 'bottom-center'`

### Chat

Complete chat UI components for building AI chat interfaces.

#### LMChatContainer
All-in-one chat interface combining message list and input.

```tsx
const chatRef = useRef<LMChatContainerRef>(null)

<LMChatContainer
  ref={chatRef}
  messages={messages}
  onSend={(content) => sendMessage(content)}
  isGenerating={isStreaming}
  onStop={() => stopGeneration()}
  placeholder="Ask anything..."
  showTypingIndicator={isStreaming}
  height="600px"
  enterToSend
/>
```

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `messages` | `ChatMessage[]` | - | **Required** message list |
| `onSend` | `(content: string) => void` | - | Send handler |
| `onStop` | `() => void` | - | Stop generation |
| `isGenerating` | `boolean` | `false` | AI generating state |
| `placeholder` | `string` | `'输入消息...'` | Input placeholder |

### AI Chat Interface
```tsx
const [messages, setMessages] = useState<ChatMessage[]>([])
const [isGenerating, setIsGenerating] = useState(false)
const chatRef = useRef<LMChatContainerRef>(null)

const handleSend = async (content: string) => {
  setMessages(prev => [...prev, { id: Date.now().toString(), role: 'user', content }])
  setIsGenerating(true)
  const response = await callAI(content)
  setMessages(prev => [...prev, { id: (Date.now() + 1).toString(), role: 'assistant', content: response }])
  setIsGenerating(false)
}

<LMChatContainer
  ref={chatRef}
  messages={messages}
  onSend={handleSend}
  isGenerating={isGenerating}
  onStop={() => setIsGenerating(false)}
  showTypingIndicator={isGenerating}
  height="100vh"
/>
```

### Theme Switching
```tsx
const [theme, setTheme] = useState<'light' | 'dark'>('light')

<div data-theme={theme}>
  <LMButton onClick={() => setTheme(t => t === 'light' ? 'dark' : 'light')}>
    Toggle Theme
  </LMButton>
</div>
```
