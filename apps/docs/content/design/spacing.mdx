# 间距系统

LumenUI 使用 4px 为基础单位的间距系统，确保视觉节奏的一致性。

## 基础间距

| 名称 | 值 | CSS 类 | 用途 |
| --- | --- | --- | --- |
| 0 | 0px | `p-0`, `m-0` | 无间距 |
| 0.5 | 2px | `p-0.5`, `m-0.5` | 微小间距 |
| 1 | 4px | `p-1`, `m-1` | 紧凑间距 |
| 2 | 8px | `p-2`, `m-2` | 小间距 |
| 3 | 12px | `p-3`, `m-3` | 常规间距 |
| 4 | 16px | `p-4`, `m-4` | 中等间距 |
| 5 | 20px | `p-5`, `m-5` | 较大间距 |
| 6 | 24px | `p-6`, `m-6` | 大间距 |
| 8 | 32px | `p-8`, `m-8` | 区块间距 |
| 10 | 40px | `p-10`, `m-10` | 大区块间距 |
| 12 | 48px | `p-12`, `m-12` | 页面级间距 |

## 圆角系统

| 名称 | 值 | CSS 类 | 用途 |
| --- | --- | --- | --- |
| none | 0px | `rounded-none` | 无圆角 |
| sm | 6px | `rounded-sm` | 小元素 |
| md | 8px | `rounded-md` | 输入框 |
| lg | 10px | `rounded-lg` | 卡片 |
| xl | 12px | `rounded-xl` | 按钮、模态框 |
| 2xl | 16px | `rounded-2xl` | 大卡片 |
| 3xl | 20px | `rounded-3xl` | 特大元素 |
| full | 9999px | `rounded-full` | 圆形、胶囊 |

## 布局间距

### 页面边距

```tsx
// 移动端
<div className="px-4">

// 平板
<div className="px-6">

// 桌面端
<div className="px-8">
```

### 区块间距

```tsx
// 紧凑
<div className="space-y-4">

// 正常
<div className="space-y-6">

// 宽松
<div className="space-y-8">
```

### 网格间距

```tsx
// 紧凑网格
<div className="grid gap-4">

// 正常网格
<div className="grid gap-6">

// 宽松网格
<div className="grid gap-8">
```

## 最佳实践

1. **保持一致性** - 同类元素使用相同的间距
2. **视觉层级** - 用间距区分内容层级
3. **响应式** - 根据屏幕大小调整间距
4. **留白** - 适当的留白能提升阅读体验
