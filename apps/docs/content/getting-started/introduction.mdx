# 介绍

LumenUI 是一个基于 React 的现代 UI 组件库，采用 Apple-like 设计风格，使用 TypeScript 和 TailwindCSS v4 构建。

## 特性

- **Apple-like 设计风格** - 精致的圆角系统、柔和的阴影效果、流畅的过渡动画
- **TypeScript 优先** - 完整的类型定义，提供优秀的开发体验
- **TailwindCSS v4** - 使用最新的 TailwindCSS 构建，样式易于定制
- **丰富的组件** - 提供 30+ 常用组件，覆盖表单、数据展示、导航、反馈等场景
- **主题系统** - 支持 5 种内置主题，通过 CSS 变量轻松定制
- **无障碍支持** - 支持 `prefers-reduced-motion`，满足 WCAG 对比度标准

## 设计理念

LumenUI 的设计灵感来源于 Apple 的设计语言，追求简洁、优雅、精致的用户体验：

- **圆角系统** - 使用 6px、10px、12px、16px、20px 的精致圆角
- **阴影效果** - 柔和的多层阴影，营造层次感
- **动画效果** - 快速流畅的过渡动画（120ms-250ms）
- **色彩系统** - 以 Apple System Blue (`#007AFF`) 为主色调

## 组件分类

LumenUI 提供以下几类组件：

| 分类 | 组件 |
| --- | --- |
| 通用 General | Button, Badge |
| 表单 Form | Input, Textarea, NumberInput, SearchInput, Select, Checkbox, Radio, Switch, Field, DatePicker, Upload |
| 数据展示 Data Display | Table, StatCard, Tooltip, Card, Tabs, Empty |
| 导航 Navigation | Menu, Dropdown, Pagination |
| 反馈 Feedback | Message, Modal, Confirm, Drawer |
| 聊天 Chat | ChatContainer, ChatList, ChatMessage, ChatBubble, ChatInput, TypingIndicator, CodeBlock, MarkdownRenderer |

## 浏览器支持

LumenUI 支持所有现代浏览器：

- Chrome >= 90
- Firefox >= 88
- Safari >= 14
- Edge >= 90
