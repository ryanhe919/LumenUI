# 安装

## 使用包管理器安装

```bash npm2yarn
npm install @ryanhe919/lumen-ui
```

## 引入样式

LumenUI 需要引入 CSS 样式文件才能正常工作。在你的应用入口文件中引入：

```tsx filename="main.tsx"
import '@ryanhe919/lumen-ui/style.css'
```

或者在 CSS 文件中引入：

```css filename="globals.css"
@import '@ryanhe919/lumen-ui/style.css';
```

## TailwindCSS 配置

如果你的项目使用 TailwindCSS，建议将 LumenUI 添加到 content 配置中：

```js filename="tailwind.config.js"
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    // ... 其他路径
    './node_modules/@ryanhe919/lumen-ui/**/*.{js,ts,jsx,tsx}',
  ],
  // ...
}
```

## TypeScript 配置

LumenUI 内置 TypeScript 类型定义，无需额外配置。

## 按需引入

LumenUI 支持 tree-shaking，你可以按需引入组件：

```tsx
// 引入单个组件
import { LMButton } from '@ryanhe919/lumen-ui'

// 或者使用子路径引入
import { LMButton } from '@ryanhe919/lumen-ui/components/General'
```

## Peer Dependencies

LumenUI 需要以下 peer dependencies：

```json
{
  "react": ">=18.0.0",
  "react-dom": ">=18.0.0"
}
```

确保你的项目已安装这些依赖。
