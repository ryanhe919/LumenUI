# Switch 开关

用于切换两种状态，如开/关、启用/禁用等。

## 引入

```tsx
import { LMSwitch } from '@ryanhe919/lumen-ui'
```

## 基础用法

<LivePreview code={`
<LMSwitch label="飞行模式" />
`} />

## 带描述

<LivePreview code={`
<LMSwitch label="深色模式" description="启用深色主题获得更好的夜间浏览体验" />
`} />

## API

### Props

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| label | 标签文字 | `string` | - |
| description | 描述文字 | `string` | - |
| checked | 开关状态 | `boolean` | - |
| onChange | 状态变化回调 | `(e: ChangeEvent) => void` | - |
| size | 尺寸 | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl'` | `'md'` |
| error | 错误状态 | `boolean` | `false` |
| errorMessage | 错误提示信息 | `string` | - |
| disabled | 禁用状态 | `boolean` | `false` |
