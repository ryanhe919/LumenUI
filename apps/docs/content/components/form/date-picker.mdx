# DatePicker 日期选择

用于选择日期的组件。

## 引入

```tsx
import { LMDatePicker } from '@ryanhe919/lumen-ui'
```

## 基础用法

<LivePreview code={`
<LMDatePicker placeholder="选择日期" />
`} />

## 尺寸

<LivePreview code={`
<div className="flex flex-col gap-3">
  <LMDatePicker size="sm" placeholder="小尺寸" />
  <LMDatePicker size="md" placeholder="中尺寸" />
  <LMDatePicker size="lg" placeholder="大尺寸" />
</div>
`} />

## 错误状态

<LivePreview code={`
<LMDatePicker
  error
  errorMessage="请选择有效日期"
  placeholder="选择日期"
/>
`} />

## 日期范围限制

使用 `minDate` 和 `maxDate` 属性限制可选日期范围。

```tsx
<LMDatePicker
  minDate={new Date('2024-01-01')}
  maxDate={new Date('2024-12-31')}
  placeholder="2024年内的日期"
/>
```

## 禁用特定日期

使用 `disabledDate` 属性禁用特定日期。

```tsx
<LMDatePicker
  disabledDate={(date) => date.getDay() === 0 || date.getDay() === 6}
  placeholder="禁用周末"
/>
```

## API

### Props

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| value | 当前值 | `Date \| null` | - |
| onChange | 值变化回调 | `(value: Date \| null) => void` | - |
| placeholder | 占位文本 | `string` | - |
| minDate | 最小日期 | `Date` | - |
| maxDate | 最大日期 | `Date` | - |
| disabledDate | 禁用日期判断函数 | `(date: Date) => boolean` | - |
| size | 尺寸 | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl'` | `'md'` |
| error | 错误状态 | `boolean` | `false` |
| errorMessage | 错误信息 | `string` | - |
| disabled | 禁用状态 | `boolean` | `false` |
