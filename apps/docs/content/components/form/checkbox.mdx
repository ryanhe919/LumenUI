# Checkbox 复选框

用于在多个选项中选择多项。

## 引入

```tsx
import { LMCheckbox } from '@ryanhe919/lumen-ui'
```

## 基础用法

<LivePreview code={`
<LMCheckbox label="同意条款和条件" />
`} />

## 带描述

<LivePreview code={`
<LMCheckbox label="邮件通知" description="接收关于账户活动和安全的邮件" />
`} />

## 禁用状态

<LivePreview code={`
<div className="flex flex-col gap-2">
  <LMCheckbox label="禁用未选中" disabled />
  <LMCheckbox label="禁用已选中" disabled checked />
</div>
`} />

## 错误状态

<LivePreview code={`
<LMCheckbox label="同意条款和条件" error errorMessage="您必须接受条款才能继续" />
`} />

## API

### Props

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| label | 标签文字 | `string` | - |
| description | 描述文字 | `string` | - |
| checked | 选中状态 | `boolean` | - |
| onChange | 状态变化回调 | `(e: ChangeEvent) => void` | - |
| size | 尺寸 | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl'` | `'md'` |
| error | 错误状态 | `boolean` | `false` |
| errorMessage | 错误提示信息 | `string` | - |
| disabled | 禁用状态 | `boolean` | `false` |
