# Select 选择器

从下拉菜单中选择一个或多个选项。

## 引入

```tsx
import { LMSelect } from '@ryanhe919/lumen-ui'
```

## 基础用法

<LivePreview code={`
<LMSelect
  options={[
    { label: '选项一', value: 'option1' },
    { label: '选项二', value: 'option2' },
    { label: '选项三', value: 'option3' },
  ]}
  placeholder="请选择"
/>
`} />

## 禁用选项

<LivePreview code={`
<LMSelect
  options={[
    { label: '选项一', value: 'option1' },
    { label: '选项二（禁用）', value: 'option2', disabled: true },
    { label: '选项三', value: 'option3' },
  ]}
  placeholder="部分选项禁用"
/>
`} />

## API

### Props

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| options | 选项列表 | `LMSelectOption[]` | `[]` |
| value | 当前值 | `string \| number \| (string \| number)[]` | - |
| onChange | 值变化回调 | `(value) => void` | - |
| placeholder | 占位文本 | `string` | `'Select...'` |
| multiple | 多选模式 | `boolean` | `false` |
| size | 尺寸 | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl'` | `'md'` |
| error | 错误状态 | `boolean` | `false` |
| errorMessage | 错误信息 | `string` | - |
| disabled | 禁用状态 | `boolean` | `false` |

### LMSelectOption

| 属性 | 说明 | 类型 | 必填 |
| --- | --- | --- | --- |
| label | 选项标签 | `string` | 是 |
| value | 选项值 | `string \| number` | 是 |
| disabled | 禁用选项 | `boolean` | 否 |
