# Radio 单选框

用于在多个选项中选择一项。

## 引入

```tsx
import { LMRadio } from '@ryanhe919/lumen-ui'
```

## 基础用法

<LivePreview code={`
<div className="flex flex-col gap-1">
  <LMRadio name="demo" value="1" label="选项一" defaultChecked />
  <LMRadio name="demo" value="2" label="选项二" />
  <LMRadio name="demo" value="3" label="选项三" />
</div>
`} />

## API

### Props

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| label | 标签文字 | `string` | - |
| description | 描述文字 | `string` | - |
| name | 单选框组名称 | `string` | - |
| value | 选项值 | `string` | - |
| checked | 选中状态 | `boolean` | - |
| onChange | 状态变化回调 | `(e: ChangeEvent) => void` | - |
| size | 尺寸 | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl'` | `'md'` |
| error | 错误状态 | `boolean` | `false` |
| errorMessage | 错误提示信息 | `string` | - |
| disabled | 禁用状态 | `boolean` | `false` |
