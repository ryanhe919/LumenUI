# Menu 菜单

用于网站导航的菜单组件。

## 引入

```tsx
import { LMMenu } from '@ryanhe919/lumen-ui'
```

## 基础用法

<LivePreview code={`
<LMMenu
  items={[
    { key: 'home', label: '首页' },
    { key: 'products', label: '产品' },
    { key: 'about', label: '关于' },
  ]}
/>
`} />

## 禁用项

<LivePreview code={`
<LMMenu
  items={[
    { key: 'home', label: '首页' },
    { key: 'disabled', label: '禁用项', disabled: true },
    { key: 'about', label: '关于' },
  ]}
/>
`} />

## 垂直模式

<LivePreview code={`
<LMMenu
  mode="vertical"
  items={[
    { key: 'home', label: '首页' },
    { key: 'products', label: '产品' },
    { key: 'about', label: '关于' },
  ]}
/>
`} />

## 子菜单

```tsx
const items = [
  { key: 'home', label: '首页' },
  {
    key: 'products',
    label: '产品',
    children: [
      { key: 'product-1', label: '产品 A' },
      { key: 'product-2', label: '产品 B' },
    ],
  },
  { key: 'about', label: '关于' },
]

<LMMenu items={items} />
```

## API

### Props

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| items | 菜单项 | `MenuItem[]` | `[]` |
| activeKey | 当前激活项 | `string` | - |
| defaultActiveKey | 默认激活项 | `string` | - |
| onChange | 切换回调 | `(key: string) => void` | - |
| mode | 菜单模式 | `'horizontal' \| 'vertical'` | `'horizontal'` |

### MenuItem

| 属性 | 说明 | 类型 | 必填 |
| --- | --- | --- | --- |
| key | 菜单项标识 | `string` | 是 |
| label | 菜单项标题 | `ReactNode` | 是 |
| icon | 菜单项图标 | `ReactNode` | 否 |
| disabled | 禁用项 | `boolean` | 否 |
| children | 子菜单项 | `MenuItem[]` | 否 |
