# Dropdown 下拉菜单

点击或悬停触发的弹出式菜单。

## 引入

```tsx
import { LMDropdown } from '@ryanhe919/lumen-ui'
```

## 基础用法

<LivePreview code={`
<LMDropdown
  items={[
    { key: 'edit', label: '编辑' },
    { key: 'copy', label: '复制' },
    { key: 'delete', label: '删除' },
  ]}
  onSelect={(key) => console.log(key)}
>
  <LMButton>更多操作</LMButton>
</LMDropdown>
`} />

## 分割线

```tsx
const items = [
  { key: 'edit', label: '编辑' },
  { key: 'copy', label: '复制' },
  { type: 'divider' },
  { key: 'delete', label: '删除', danger: true },
]
```

## API

### Props

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| items | 菜单项 | `DropdownItem[]` | `[]` |
| onSelect | 选择回调 | `(key: string) => void` | - |
| trigger | 触发方式 | `'click' \| 'hover'` | `'click'` |
| placement | 弹出位置 | `'bottomLeft' \| 'bottomRight' \| 'topLeft' \| 'topRight'` | `'bottomLeft'` |
| disabled | 禁用 | `boolean` | `false` |
| children | 触发元素 | `ReactElement` | - |

### DropdownItem

| 属性 | 说明 | 类型 | 必填 |
| --- | --- | --- | --- |
| key | 菜单项标识 | `string` | 是（非分割线时） |
| label | 菜单项标题 | `ReactNode` | 是（非分割线时） |
| icon | 菜单项图标 | `ReactNode` | 否 |
| disabled | 禁用项 | `boolean` | 否 |
| danger | 危险操作 | `boolean` | 否 |
| type | 类型 | `'divider'` | 否 |
