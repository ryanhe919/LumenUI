# LumenUI

基于 React 的现代 UI 组件库，采用 Apple-like 设计风格，使用 TypeScript 和 TailwindCSS v4 构建。

## 特性

- **Apple-like 设计风格** - 精致的圆角、柔和的阴影、流畅的动画
- **TypeScript 优先** - 完整的类型定义，提供优秀的开发体验
- **主题定制** - 支持 5 种内置主题，通过 CSS 变量轻松定制
- **丰富组件** - 30+ 常用组件，覆盖各种场景

## 快速开始

```bash
npm install @ryanhe919/lumen-ui
```

```tsx
import { LMButton } from '@ryanhe919/lumen-ui'
import '@ryanhe919/lumen-ui/styles.css'

function App() {
  return <LMButton>Hello LumenUI</LMButton>
}
```
